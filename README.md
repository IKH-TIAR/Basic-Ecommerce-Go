# hello-go

A simple Go project.

## Getting Started

1. **Build the project:**
   ```sh
   go build
   ```
2. **Run the project:**
   ```sh
   go run main.go
   ```


## Routes

| Method | Route        | Description                       |
|--------|--------------|-----------------------------------|
| GET    | /            | Test route, returns 'Test Route'   |
| GET    | /products    | Returns list of products (JSON)    |

## Project Structure
- `main.go`: Main application file
- `go.mod`: Go module definition

## Requirements
- Go 1.18 or newer
